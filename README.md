# First_time_web
## ターミナルからVScodeを起動する時

```
code .
```
## コミット方法
```
git commit -m "変更した部分の文章を打つ"
```
## プッシュ方法
プッシュとはリモート（サーバーex:GitHub）のブランチの更新
```bash
git push origin <リモート上のブランチ名>

# mainにプッシュする場合は以下のようにする
　git push origin main
```
## 新規ファイル作成方法
```bash
touch <作りたいファイルの名前>

# index.htmlを作りたい場合は以下のようにする
  touch index.html
```
## ファイルの削除方法
```
rm <削除したいファイル名>
```
